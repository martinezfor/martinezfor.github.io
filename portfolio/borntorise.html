A 2,5D platform metroidvania project made with Unreal Engine 4.27.2

Project information

  • Category: Unreal Engine Game
  • Project date: 11 July 2023
  • Download page: Steam Page
  • Made by: Raccoons Making Games


Description

    This was the final project of the HND in Videogames Programming at ESAT. It is a 2.5D platformer that was developed by a multidisciplinary team of 7 programmers, 6 artists and 3 game designers.

    This project tried to be a very close approximation of how a video games studio works. We had tight schedules with deadlines and milestones such as prototype and first playable. We worked under an adaptation of the scrum methodology where we had weekly sprints.

    The most important aspect of this project is the communication both between peers of your own team and between teams. Constant communication with our fellow designers and artist is key to have a smooth development and to better iterate on the game.

My Rol in this Project

  • Worked as an AI programmer for the enemies using C++ and Unreal Engine
  • Programmed Bouncing hazard
  • Technical artist (Animations, UI)
  • Worked in a team of 15 people
  • Used Perforce as a source control tool
